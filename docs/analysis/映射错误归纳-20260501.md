## 已知错误

### 1. WTI原油失守101美元 → 映射到 Healthcare

**时间**: 2026-05-01 21:55
**事件**: WTI原油失守101美元/桶，日内跌3.94%
**AI 判断**: ✅ 正确（energy 事件，hit）
**系统映射**: ❌ Healthcare

**根因**: `configs/conduction_chain.yaml`

**(a) 关键词不分方向（第 330 行）**
```yaml
- event_keywords: ["原油", "能源价格", "大宗商品", "供应链中断"]
    chain_id: "inflation_shock_chain"
```
原油涨价和跌价都匹配同一条链。WTI 跌 3.94%（通缩），但走了通胀链。

**(b) 通胀链 sectors 包含 Healthcare（第 109 行）**
```yaml
sectors:
  - name: "Consumer Staples"
  - name: "Utilities"
  - name: "Healthcare"   # ← 油价下跌跟医疗无关
```

### 2. 现货黄金突破4640美元 → 未映射任何板块

**时间**: 2026-05-01 22:02
**事件**: 现货黄金突破4640美元/盎司，日内涨0.41%
**AI 判断**: ✅ 正确（黄金/commodity，hit）
**系统映射**: ❌ 无板块映射

**根因**: `configs/conduction_chain.yaml`

**关键词缺失**：传导链所有关键词列表均未包含"黄金"/"贵金属"/"gold"。唯一相关的只有"大宗商品"（第 330 行），但这条关键词触发的是 `inflation_shock_chain`（通胀链），映射到 Consumer Staples/Utilities/Healthcare，也不是黄金应有的板块（Materials/贵金属板块）。
