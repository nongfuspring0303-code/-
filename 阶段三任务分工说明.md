# Phase 3 任务分工说明

**项目**: EDT指挥中心 - 网页可视化投研系统  
**核心理念**: 一个网页看全链路，新闻变→板块变→股票变→决策变  
**变更ID**: phase-3-command-center

---

## 一、用户故事

| 用户操作 | 系统响应 |
|----------|----------|
| 打开网页 | 显示实时新闻流（左栏） |
| 点击新闻 | 中栏显示受影响板块，右栏显示机会池 |
| 点击板块 | 右栏筛选该板块的优质股票 |
| 查看机会 | 显示入场区间、风险旗标、最终动作 |

---

│       ↑                   ↑                   ↑                │

---

## 三、分工原则

- 三人三模块，**互不依赖**，**并行开发**
- 通过**事件契约**交互，不存在代码级依赖
- 各自独立开发、独立测试，最后通过WebSocket集成
- **任何字段变更必须四联动**: schemas + tests + module-registry.yaml + 文档

---

## 四、任务清单

### 模块A：后端数据流（新闻→板块）

| 任务ID | 交付物 | 验收标准 |
|--------|--------|----------|
| A-1 | `configs/sector_impact_mapping.yaml` | 能输出板块方向与影响强度 |

| A-3 | `configs/conduction_chain.yaml` | 可展示"宏观→板块→主题"链路 |

### 模块B：策略层（板块→股票→机会）

| 任务ID | 交付物 | 验收标准 |
|--------|--------|----------|
| B-1 | `configs/premium_stock_pool.yaml` + 筛选逻辑 | 机会池仅输出优质池内股票 |
| B-2 | `opportunity_score` 规则与排序逻辑 | 利好/利空输入输出可区分(LONG/SHORT/WATCH) |
| B-3 | 机会卡字段(入场区间/风险旗标/理由摘要) | 右栏卡片可直接用于投研讨论 |

### 模块C：前端可视化 + 实时总线 + 风控

| 任务ID | 交付物 | 验收标准 |
|--------|--------|----------|
| C-1 | `event_update`、`sector_update`、`opportunity_update` schema | 前端可稳定订阅并渲染 |
| C-2 | `canvas/index.html` + `styles.css` + `app.js` 三栏联动界面 | 点击新闻后中右栏按trace_id联动 |
| C-3 | 高风险拦截与 `PENDING_CONFIRM` 校验 | 未确认请求不得进入执行 |
| C-4 | WebSocket事件总线 `event_bus.py` | 消息按trace_id有序，断线自动重连，支持重放 |
| C-5 | 时间线回放功能 | 可拖拽回看历史 |

## 五、模块A：后端数据流（新闻→板块）

### 职责
接收新闻，输出受影响的板块及方向（LONG/SHORT）

### 输入
- 新闻源API（路透/新浪/华尔街见闻）

### 输出（给事件总线）
```json
{
  "type": "sector_update",
  "trace_id": "evt_xxx",
  "sectors": [
    {"name": "半导体", "direction": "LONG", "impact_score": 0.85},
    {"name": "新能源", "direction": "SHORT", "impact_score": 0.40}
  ],
  "conduction_chain": [
    {"level": "macro", "name": "原油", "relation": "affects_above"},
    {"level": "sector", "name": "能源", "relation": "same"},
    {"level": "theme", "name": "化工", "relation": "affects_below"}
  ],
  "timestamp": "2026-04-03T10:32:00Z"
}
```

### 复用Phase 2模块
- `ai_event_intel.py`: 新闻摄取
- `intel_modules.py`: 情报处理
- `narrative_state_recognizer.py`: 叙事状态识别
- `conduction_mapper.py`: 传导映射

### 验收
- [ ] 新闻点击后，板块更新延迟 ≤1s
- [ ] 方向明确（LONG/SHORT，不允许只有分数）
- [ ] 传导链完整展示（宏观→板块→主题）
- [ ] 同事件30分钟内不重复显示
- [ ] trace_id全程贯穿

## 六、模块B：策略层（板块→股票→机会）

### 职责
接收板块方向，输出优质股票及建仓机会

### 输入
- A模块输出的 `sector_update`

### 输出（给事件总线）
```json
{
  "type": "opportunity_update",
  "trace_id": "evt_xxx",
  "opportunities": [
    {
      "symbol": "688981",
      "name": "中芯国际",
      "sector": "半导体",
      "signal": "LONG",
      "entry_zone": {"support": 45.2, "resistance": 47.5},
      "risk_flags": [{"type": "liquidity", "level": "medium", "description": "流动性偏低"}],
      "final_action": "EXECUTE",
      "reasoning": "半导体板块强利好，个股基本面评分0.82，入场区间合理",
      "confidence": 0.85,
      "timestamp": "2026-04-03T10:32:05Z"
    }
  ]
}
```

### 机会卡字段说明
| 字段 | 说明 | 示例 |
|------|------|------|
| symbol | 股票代码 | 688981 |
| name | 股票名称 | 中芯国际 |
| sector | 所属板块 | 半导体 |
| signal | 信号方向 | LONG/SHORT/WATCH |
| entry_zone.support | 支撑位 | 45.2 |
| entry_zone.resistance | 阻力位 | 47.5 |
| risk_flags | 风险旗标数组 | [{"type":"liquidity", "level":"medium"}] |
| final_action | 最终动作 | EXECUTE/WATCH/BLOCK/PENDING_CONFIRM |
| reasoning | 理由摘要 | 50字以内 |
| confidence | 置信度 | 0.85 |

### 复用Phase 2模块
- `market_validator.py`: 市场验证
- `signal_scorer.py`: 信号评分
- `ai_signal_adapter.py`: AI信号适配
- `decision_card.py`: 决策卡生成
- `fatigue_calculator.py`: 疲劳度计算

### 验收
- [ ] 板块切换后，机会池更新延迟 ≤1s
- [ ] 利好→LONG，利空→SHORT（差异化≥80%）
- [ ] 机会池只显示优质池内股票
- [ ] 机会卡字段完整（入场区间+风险旗标+理由）
- [ ] 高风险自动BLOCK或PENDING_CONFIRM

## 七、模块C：前端可视化 + 实时总线 + 风控

### 职责
- WebSocket事件总线
- 三栏联动网页（新闻/板块/机会池）
- 风控校验

### 输入
- 订阅 A模块的 sector_update
- 订阅 B模块的 opportunity_update

### 页面设计

#### 左栏：新闻雷达
- 实时滚动新闻（WebSocket推送）
- 颜色标记严重度（红黄蓝绿）
- 点击新闻高亮，联动中右栏

#### 中栏：板块热力图
- 颜色语义：绿=利好，红=利空，灰=中性
- 数值显示影响强度
- 传导链层级展示

#### 右栏：机会池
- 只显示优质池内股票
- LONG/SHORT/WATCH分区
- 入场区间、风险旗标
- 执行按钮（需人工确认）

### 配色方案
- 背景：#1a1a2e（深蓝黑）
- 面板：#16213e（深蓝）
- 利好：#00d26a（绿色）
- 利空：#ff4757（红色）
- 中性：#636e72（灰色）
- 文字：#dfe6e9（浅灰白）
- 高亮：#feca57（黄色）

### 交付物
| 文件 | 说明 |

| `canvas/index.html` | 三栏联动主页面 |

### WebSocket服务
- 地址：`ws://localhost:8765`
- 消息按trace_id分组有序
- 断线自动重连
- 消息持久化支持重放

### 风控规则
```python
# 高风险拦截逻辑
def risk_check(opportunity):
    # 高风险自动BLOCK
    if len(opportunity.risk_flags) >= 3:
        return "BLOCK"
        
    # 中风险进入待确认
    if any(f.level == "high" for f in opportunity.risk_flags):
        return "PENDING_CONFIRM"
        
    # AI超时/错误触发G7降级
    if opportunity.ai_timeout_or_error:
        return "WATCH"
        
    return opportunity.final_action
```

## 八、事件契约（三人对接标准）

### 三类消息
| 消息类型 | 字段 | 说明 |
|----------|------|------|
| event_update | trace_id, headline, source, severity, evidence_score, narrative_state | 新闻事件 |
| sector_update | trace_id, sectors[].name/direction/impact_score/confidence, conduction_chain | 板块更新 |
| opportunity_update | trace_id, opportunities[].symbol/name/signal/entry_zone/risk_flags/final_action/reasoning | 机会更新 |

### 关键规则
1. 同一 `trace_id` 的消息按时间有序
2. A/B模块只管发消息，不关心谁接收
3. C模块负责订阅、路由、推送
4. **任何字段变更必须四联动**: schemas + tests + module-registry.yaml + 文档

---

## 九、验收标准汇总

| 维度 | 指标 | 目标 |
|------|------|------|
| **可视化** | 联动延迟 | 新闻→板块 ≤1s, 板块→机会 ≤1s |
| **可视化** | 状态动画 | WATCH→EXECUTE有过渡效果 |
| **功能** | 决策分化 | 利好/利空差异化率 ≥80% |
| **功能** | 机会池 | 仅显示优质池内股票 |
| **风控** | 拦截率 | 高风险100% BLOCK/PENDING_CONFIRM |
| **可靠** | trace_id | 贯穿率100% |
| **可靠** | 断线恢复 | 自动重连 + 消息重放 |
| **可靠** | 回放 | 支持按trace_id回放历史 |

## 十、对接示例

### A模块发送
```python
await event_bus.publish("sector_update", {
    "trace_id": "evt_20260403_1032",
    "sectors": [
        {"name": "半导体", "direction": "LONG", "impact_score": 0.85, "confidence": 0.90}
    ],
    "conduction_chain": [...],
    "timestamp": "2026-04-03T10:32:00Z"
})
```

### B模块发送
```python
await event_bus.publish("opportunity_update", {
    "trace_id": "evt_20260403_1032",
    "opportunities": [
        {
            "symbol": "688981",
            "name": "中芯国际",
            "signal": "LONG",
            "entry_zone": {"support": 45.2, "resistance": 47.5},
            "risk_flags": [{"type": "liquidity", "level": "medium"}],
            "final_action": "EXECUTE",
            "reasoning": "半导体板块强利好，个股基本面评分0.82"
        }
    ],
    "timestamp": "2026-04-03T10:32:05Z"
})
```

### C模块接收
```javascript
ws.onmessage = (e) => {
    const msg = JSON.parse(e.data);
    switch(msg.type) {
        case 'event_update':
            renderNews(msg.data);
            break;
        case 'sector_update':
            renderSectors(msg.data);
            break;
        case 'opportunity_update':
            renderOpportunities(msg.data);
            break;
    }
}
```
