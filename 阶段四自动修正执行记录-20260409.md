## 7. 增量执行记录（2026-04-09 晚间）

### 7.1 问题与根因
- 现象：网页可见 `Energy` 板块热力图，但机会卡为空。
- 根因：机会评分回退选股依赖板块名一致，输入为英文 `Energy`，股票池为中文 `能源`，导致 fallback `pick_by_sector()` 不命中。

### 7.2 落地修复
- 新增配置：`configs/sector_aliases.yaml`（板块别名字典，配置化）。
- 代码改造：`scripts/opportunity_score.py`
  - 新增 `SectorAliasResolver` 加载并解析别名。
  - `PremiumStockPool.pick_by_sector()` 使用 canonical sector 匹配。
  - `OpportunityScorer.build_opportunity_update()` 对 `sectors` 与 `stock_candidates` 统一 canonical key。
- 回归测试：`tests/test_opportunity_score.py::test_fallback_pool_supports_sector_alias_dictionary`

### 7.3 验证证据
- 测试：
  - `python3 -m pytest tests/test_opportunity_score.py -k sector_alias_dictionary -v` -> `1 passed`
  - `python3 -m pytest tests/test_conduction_mapper_dynamic.py tests/test_opportunity_score.py tests/test_realtime_news_monitor.py -v` -> `16 passed`
- 运行证据：
  - `logs/event_bus_history.jsonl` 最新 `opportunity_update` 已含 `symbol: XOM`、`sector: Energy`、`final_action: WATCH`。
- 结论：
  - `Energy -> XOM` 端到端链路恢复，页面可见候选股票。
